# Chat UI & jathakam page layout — prototype

Throwaway HTML/CSS/JS prototype for [ticket 015](../../docs/wayfinder/engineering/tickets/015-chat-ui-jathakam-layout.md). Two mobile mockups, styled to match the locked home-screen chrome (`spikes/practice-card-prototype/`):

- **Jathakam** — chart as hero (tap-to-reveal), panchangam block labeled "at birth", mahadasha list with tap-to-expand on the current period, sticky disclaimer.
- **Chat** — header AI marker, streaming reply with inline tap-for-grounding citation (in-flow reveal, not a floating popover — an earlier floating version overflowed the narrow phone width), violet send button, sticky disclaimer.

Open `index.html` directly in a browser. No build, no deps.
